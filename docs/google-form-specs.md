# 구글폼 문항 명세 (운영자용)

두 폼을 구글폼에서 새로 만들고, 아래 문항을 순서대로 입력한다.
완성되면 각 폼의 **게시 URL**과 **기수 prefill 키**를 안윤수에게 전달 → `config.js`에 연결한다.

## 공통 설정
- 설정 → 응답 → "이메일 주소 수집" 끄기(이름·기수로 식별).
- 각 폼 첫 페이지 상단 설명에 개인정보 안내 1줄: "수집한 정보는 40주년 행사 운영 목적에 한해 사용하며, 행사 종료 후 파기합니다."
- "기수" 문항은 prefill 대상 → 만든 뒤 "사전 채워진 링크 가져오기"로 `entry.숫자` 값을 확인해 전달.

## 폼 ① 동잠(단체복) 주문

1. (필수) **단체복 구매 여부** — 객관식: `구매합니다.` / `구매하지 않습니다.`
   - "구매하지 않습니다." 선택 시 → "양식 제출" 섹션으로 이동(이후 문항 생략).
2. **기수** — 단답형. (prefill 대상)
3. **이름(본명)** — 단답형.
4. **성함(한자)** — 장문형. 안내: "오른쪽 손목 부근에 새겨질 예정입니다. 한자 표기를 적어주세요. 구매하시는 분은 꼭 답변 부탁드립니다."
5. **성함(영문)** — 단답형. 안내: "왼쪽 손목 부근에 새겨질 예정입니다. 예: Y.S. An. 구매하시는 분은 꼭 답변 부탁드립니다."
6. (필수) **사이즈** — 객관식: `XS` / `S` / `M` / `L` / `XL` / `2XL` / `3XL`.
7. (필수) **수량** — 단답형(숫자).
8. (필수) **수령 방법** — 객관식: `행사장 수령` / `택배(주소 별도 안내)`.
9. **연락처** — 단답형(휴대폰).
10. **비고** — 장문형(선택).

마지막 섹션 설명에 결제 안내: "입금은 주문 수합 후 총액이 확정되면 별도로 안내드립니다."

## 폼 ② 동문 근황·연락처 업데이트

### 섹션 A — 연락처 (주소록용)
1. **기수** — 단답형. (prefill 대상)
2. **이름** — 단답형.
3. **휴대폰** — 단답형.
4. **이메일** — 단답형.
5. **주소·근황 한마디** — 장문형(선택).
6. (필수) **주소록 공개 범위** — 객관식: `동문에게만 공개` / `비공개`.
7. (필수) **개인정보 수집·이용 동의** — 체크박스: `동의합니다`(필수).

### 섹션 B — 졸업 후 진로 및 경력 (40년 통계용)
섹션 설명: "이 섹션의 데이터는 나중에 '40년 통계' 시각화에 사용됩니다."
1. (필수) **최종 학력** — 객관식: `학사` / `석사` / `박사` / `포닥` / `기타`.
2. (필수) **전공 분야** — 단답형. 안내: "석/박사의 경우 세부 전공 기입."
3. (필수) **현재 소속 및 직위** — 단답형. 안내: "예 - OO전자 책임연구원, OO대학교 교수, 벤처 창업 등."
4. (필수) **주요 경력 요약** — 장문형. 안내: "선배님들의 발자취를 40주년 기념 자료에 담고자 합니다. 주요 이력을 자유롭게 적어주세요."

# 설문·결제 시스템 — 회의 결정 시트

회의에서 **아래 3개만 정하면** 구현 계획을 뽑을 수 있습니다. (상세 근거: `docs/specs/2026-06-17-survey-payment-system-design.md`)

> 하려는 것: 참석조사·동잠구매 폼을 **사이트 안에서** 받고, **결제까지 자동 대조**(누가 냈는지 시스템이 매칭). 구글폼/수동대조 대체.

---

## 결정 1 — 백엔드 어디에 올리나? (전산팀장)

폼 제출을 받고 결제 웹훅을 처리할 곳이 필요합니다.

- [ ] **AWS** (전산팀 관리) — 권장. 개인정보 통제·확장성 우수, 멱등성 깔끔.
- [ ] **Google Apps Script + 구글시트** — 무료·무서버. 우리 규모엔 충분하나 동시폭주·개인정보 면에서 약함.

**물어볼 것**: AWS에 작은 백엔드(API 3개 + DB) 올릴 수 있나? 형태는(Lambda+RDS 등)? 유지보수는 전산팀이? 도메인·HTTPS 발급 가능?

---

## 결정 2 — 결제수단 (회계/총무)

세 길 다 **비영리 사단법인 가능** 확인됨. 하나 고르면 됩니다.

- [ ] **토스페이먼츠** — 비영리기관 가맹 **공식 확인**. 가장 무난.
- [ ] **PayApp** — 자동대조 기술은 최적이나 **비영리 가맹 가능한지 직접 문의 필요**.
- [ ] **이벤터스(폴백)** — 가입 막히면 즉시 이걸로. **고유번호증만으로 정산** 가능. 결제·대조를 플랫폼이 대행(결제화면 디자인만 양보).

**확인할 서류**: 우리 사단법인 **사업자등록증 / 고유번호증** 중 무엇을 갖고 있나? 정산받을 **법인 통장**?

---

## 결정 3 — 참가비 세무 (회계/세무)

- [ ] 참가비(대가성)가 **수익사업 과세 대상인지** 세무 확인.
- 참고: 고유번호증만으론 세금계산서 발행 불가. 기부금영수증은 공익법인 지정 필요(임의단체 불가).

---

## 반드시 지켜야 할 것 (개인정보보호법 — 협상 대상 아님)

신청 폼에 개인정보(이름·연락처)를 받으므로 동아리도 법적 의무가 있습니다. 구현에 자동 포함 예정:

- 동의 문구 4항목(목적·항목·보유기간·거부권) + 동의 체크
- 사이트에 **개인정보 처리방침** 게시
- 접근권한 통제 + **행사 종료 후 파기**
- (미이행 시 과태료: 처리방침 1천만 / 미파기 3천만 이하)

## 추가 안건 — 역대 영상 호스팅 (저작권) 🟡

역대 wmv 25편을 mp4로 변환 완료(**총 24GB**, 편당 1~2GB). 갤러리 영상 탭에 올릴지/어디에 올릴지 미정. `data/videos.json`은 현재 비어 있어 라이브엔 영상 없음.

**문제**: 상당수가 국제대회·방송 성격(`Micro-Robot World Cup`, `FIRA France98 Highlights`, `98 아시아 태평양 대회` 등) → 저작권 권리자 불명/외부. **어떤 플랫폼도 합법화는 못 함.** 레버는 ① 자동 저작권 스캔 회피 ② 공개 노출 최소화 둘뿐.

- 유튜브(한정공개 포함)는 **Content ID 전수 스캔** → 클레임·strike로 동아리 채널 정지 위험. 비추천.

**고를 안 (하나):**
- [ ] **R2 자체 호스팅 + 회원 전용 (권장)** — 720p 재인코딩(24GB→3~6GB) 후 Cloudflare R2(전송 무료, 저장 ~$0.4/월)에 올려 회원 게이트 뒤에서 재생. 자동스캔 없음·노출 최소. (게이트는 페이지 차단+비공개 URL 수준, 철벽 아님 → 명백 방송분은 스틸 권장)
- [ ] **하이브리드** — 자체 촬영분(총회·라인트레이서 등 8편)만 유튜브 한정공개, 방송·대회 의심분은 **영상 없이 스틸컷만** 전시.
- [ ] **전부 스틸만 + 영상 비공개 보관** — 가장 안전(리스크 0). 웹엔 영상 스틸 504장만.
- [ ] **Cloudflare Stream** — 적응형 스트리밍 품질↑, 저장+전송 과금↑. 자동스캔 없음.

**자체 촬영분 8편 초안**: `data/videos.draft.json` (업로드 후 영상 ID만 채우면 됨). 분류 근거: 자체=동아리 총회·교내 대회 / 방송의심=국제 월드컵·FIRA Highlights.

**회의 후 알려줄 한 줄**: "영상 = (R2회원 / 하이브리드 / 스틸만 / Stream)"

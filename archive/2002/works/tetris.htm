밍밍 for game

MR 02 김민규

 [밍밍 for game]의 특징 :


1. Code memory 8k, Data memory 16k, Flash memory 16k를 내장하고 있어서 데이터가 큰 프로그램도 짜넣을 수 있음.

        - 현재는 테트리스와 벌레 게임이 프로그램 돼 있음


2. dip스위치가 달려있어서 게임의 mode를 선택할 수 있음.


3. Back light가 내장된 LCD module을 사용해서 어두운 곳에서도 게임이 가능함.


4. LCD에 표현된 모든 내용을 RAM에 저장하고 있어서 LCD에 출력하는 속도가 빠름.

 영상 자료

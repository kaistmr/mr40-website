# 드라마 「카이스트」 SBS 무료 VOD 링크 카드 — 설계

날짜: 2026-06-11
상태: 승인됨 (사용자 확인)

## 목적

연혁(timeline.html)의 1998. 1 항목("SBS 드라마 카이스트의 모델 동아리로 지명")에
SBS 공식 무료 다시보기로 연결되는 썸네일 링크 카드를 붙인다.
드라마 영상을 직접 호스팅/다운로드하지 않고(저작권), 공식 페이지로 보낸다.

## 배경·결정 사항

- SBS는 유튜브식 외부 임베드 플레이어를 제공하지 않음 → iframe 임베드 대신 **링크 카드** 방식.
- 링크 대상은 특정 회차가 아닌 **무료보기 목록 페이지** (링크 부패 위험 최소):
  `https://programs.sbs.co.kr/drama/kaist/vods/59992`
- 썸네일은 SBS 공식 og:image를 **핫링크** (우리 서버에 이미지 미호스팅 → 저작권상 깨끗):
  `https://img2.sbs.co.kr/img/sbs_cms/PG/2015/10/22/PG40765071_w640_h360.jpg`
- 썸네일 로드 실패 시 `onerror`로 이미지를 숨기고 텍스트 버튼만 남김.

## 변경 파일

### 1. `data/timeline.json`

타임라인 항목에 선택 필드 `link` 추가. 1998. 1 항목만 사용:

```json
"link": {
  "url": "...",
  "label": "SBS 공식 무료 다시보기",
  "image": "..."
}
```

스키마 변화는 추가 필드뿐이므로 기존 렌더링과 하위 호환.
timeline.json 소비자는 timeline.html 하나뿐임을 확인함.

### 2. `timeline.html`

- `render()`: `it.link`가 있으면 `.tl-card` 안에 미디어 블록 추가
  - `<a target="_blank" rel="noopener">` + 썸네일 + ▶ 재생 오버레이 + 라벨 버튼
  - URL·라벨·이미지 경로 모두 `escapeHtml`로 이스케이프
- CSS 약 30~40줄: 모던/레트로(터미널 그린) 양쪽 시대 톤에 맞는 카드 스타일.
  1998은 스크롤상 레트로 구간이므로 레트로 톤이 주로 노출됨.

## 범위 외 (YAGNI)

- iframe 직접 임베드 (SBS 미지원)
- 회차별 큐레이션·로봇축구 회차 특정
- 드라마 스틸컷/영상 호스팅

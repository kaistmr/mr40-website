# MR 40주년 기념 홈페이지 설계 스펙

- 날짜: 2026-06-10
- 행사일: 2026-11-28 (사이트 오픈 목표: 행사 당일까지)
- 작업 영역: `40th_website/` (외부 폴더는 읽기 전용 — CLAUDE.md 참조)

## 1. 목적과 성공 기준

KAIST 동아리 MR(Micro Robot, 1986 창립) 40주년 기념 특별 홈페이지.
동문·현역이 40년 역사를 돌아보고(타임라인·사진관·회지), 서로 연결되고(주소록),
행사에 참여(행사 안내·방명록)하게 한다.

성공 기준:
- 11/28 행사 당일 전 기능 동작
- 모바일에서 쾌적하게 동작 (동문 대부분 폰 접속 가정)
- 행사 후 관리자 없이도 수년간 죽지 않는 구조 (서버리스, 의존성 0)

## 4. 사이트맵과 페이지별 스펙

```
├── index.html        메인
├── timeline.html     연혁 타임라인
├── members.html      주소록
├── gallery.html      사진관 (사진 + 영상 탭)
├── magazine.html     회지 아카이브
├── guestbook.html    축하 방명록
├── event.html        행사 안내
├── assets/           공통 CSS·JS·로고
├── data/             gallery.json, videos.json, timeline.json
├── thumbs/           갤러리 썸네일 WebP
└── config.js         시트 CSV URL, 폼 URL, R2 베이스 URL, 접속 코드 해시
```

- **index.html**: 히어로("로봇과 함께한 마흔 해", 1986→2026), 행사 D-day 카운터, 섹션 입구 카드 6개
- **timeline.html**: 세로 스크롤 타임라인. 데이터 소스: 기존 DB `history` 테이블 111건 (Website_Server_Backup/mr_homepage_dump.sql에서 추출, 읽기만) + 회지·대회 자료 보강. 연대별 구간마다 대표 사진·영상 임베드
- **members.html**: 접속 코드 게이트 → 시트 CSV 로드 → 이름/기수/소속/관심사 검색·태그 필터 (CBSH index.html 패턴 재사용). 필드: 이름, 기수(입학년도), 소속, 이메일, 전화(선택), 관심사, 한마디
- **gallery.html**: 연도별 필터 칩 + 썸네일 그리드(lazy load) → 라이트박스(R2 원본). "영상" 탭: YouTube 임베드 목록(videos.json: id, 제목, 연도, 설명)
- **magazine.html**: 회지 표지 그리드 → 클릭 시 PDF 새 탭(브라우저 내장 뷰어). 대상: 1994 준비호, 1996 큰마음작은이야기, 1998 형배고파요, 회지 2호, 대회 자료집 등 (Donated materials 변환본)
- **guestbook.html**: 상단 "축하 한마디 남기기" 버튼(구글폼) + 시트 CSV에서 메시지 카드 렌더 (이름, 기수, 메시지)
- **event.html**: 11/28 행사 일정·장소·프로그램·참가 신청(구글폼 링크)

## 5. 데이터 흐름

```
[구글폼] 주소록·방명록 응답 → [구글시트 '웹에 게시' CSV] → JS fetch → 렌더
                                (localStorage 캐시 + 백그라운드 갱신, CBSH 패턴)

[로컬 Python 스크립트] (40th_website/scripts/, 표준 도구 + Pillow)
  원본 사진 (cd files·Donated·GoogleDrive, 읽기만)
    → 큐레이션 목록 기반 → 썸네일 WebP + thumbs/ 커밋
    → 원본급 리사이즈 → R2 업로드
    → data/gallery.json (파일명, 연도, 설명, R2 키)

[YouTube] wmv → 업로드(한정공개) → data/videos.json에 ID 수동 기재
```

## 6. 에러 처리

- 시트 CSV fetch 실패: localStorage 캐시본 표시, 캐시도 없으면 "불러올 수 없습니다 + 관리자 문의" 안내
- R2 이미지 로드 실패: 썸네일로 대체 표시
- 접속 코드 오입력: 안내 메시지, 시도 제한 없음(클라이언트 한계 인지)
- JS 비활성 환경: noscript 안내
